Downloads

A fully functional 30-day trial version of Orthopaedic Studio can be installed using the built-in ‘Plugins manager’ which is found in the Plugins menu in OsiriX. Click the downloads tab in the Plugins Manager and then select to download and install Orthopaedic Studio.

Alternatively, the 30-day trial version of Orthopaedic Studio can be downloaded manually (OrthopaedicStudio_v1.2.1.zip, 1 MB). Double-click the file in an Finder window to unzip it. Then double-click the unzipped file and follow the on-screen instructions to install it.

Requirements

In order to use Orthopaedic Studio you need a computer running Mac OS X 10.6 (Snow Leopard) or 10.7 (Lion). You also need to have a fairly recent version of OsiriX installed.

If you are interested in a version of Orthopaedic Studio running on Mac OS X 10.5 (Leopard) please contact us.
